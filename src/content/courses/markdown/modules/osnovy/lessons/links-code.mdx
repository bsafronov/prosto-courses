## Ссылки соединяют материалы

Текст ссылки заключают в квадратные скобки, а адрес — в круглые:

```md
[Документация Astro](https://docs.astro.build/)
```

Осмысленный текст ссылки полезнее фразы «нажмите здесь»: читатель заранее понимает, куда перейдёт.

## Код сохраняет точное написание

Короткую команду или имя файла обрамляют обратными кавычками: `pnpm build`. Для нескольких строк используют блок из трёх обратных кавычек и при необходимости указывают язык.

````md
```js
console.log("Привет!");
```
````
