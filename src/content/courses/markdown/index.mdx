Markdown помогает сосредоточиться на смысле текста, не отвлекаясь на панели форматирования. Курс знакомит с основными конструкциями и сразу предлагает применить их на практике.

Для занятий нужен только текстовый редактор. Примеры можно копировать в любой сервис, который поддерживает Markdown.
