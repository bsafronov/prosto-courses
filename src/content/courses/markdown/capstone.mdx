Подготовь в Markdown инструкцию для реальной повторяющейся задачи. Читатель должен понять назначение документа, быстро найти шаги и перейти к дополнительному источнику.

Перед завершением просмотри исходный текст: структура, ссылки и фрагменты кода должны быть однозначны даже без визуального редактора.
