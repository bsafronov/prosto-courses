## Заголовки создают структуру

Количество символов `#` в начале строки задаёт уровень заголовка:

```md
# Название документа
## Большой раздел
### Подраздел
```

Уровни лучше использовать последовательно. После заголовка второго уровня обычно идёт заголовок третьего, а не четвёртого.

## Выделение помогает расставить акценты

Одинарные звёздочки создают *курсив*, двойные — **сильное выделение**:

```md
Это *важное* слово, а это **главный вывод**.
```

## Списки группируют связанные мысли

Для неупорядоченного списка поставь дефис перед каждым пунктом. Для последовательности используй числа.

```md
- тезис
- пример
- вывод

1. Открой файл
2. Добавь заголовок
3. Сохрани результат
```
